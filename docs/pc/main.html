Garçounnet 2

(a silly 2 player 2d shooter using Megaman sprites)

MS-DOS executable with source code: ga2.zip
Read main source code in browser.

In order to compile this piece of DOS code, you need the Allegro game library and an allegro-friendly compiler, such as the free DJGPP C compiler.

Controls: Blue player-
Movement: arrows
Shoot: left control

Red player-
Movement: joystick axes
Shoot: firs joystick button

General controls: resume action after a point is made: R
Quit: ESC

Joystick support used to work when I had Windows 98SE. Under WinXP, it doesn't seem to work. You might have better luck with me with the newest Allegro version. There is no interface whatsoever in the game, it was just a study of sprite movement/animation, bullet limitations (each player can have at most 3 bullets on screen at any time) and simple game mechanics in general.
